Breeze Sync

Data is always saved to local storage when the user is offline or online. The user decides when to sync the data by using the sync button.

Sync flow

  • First; Any changes are saved to the server.
  • Then; any records found in the cache that is also in the tombstones is deleted where the modified time stamp is bigger than the current modified time stamp.
  • Then; for each table, all records are pulled from the server where the modified time stamp is bigger than the current modified time stamp.
